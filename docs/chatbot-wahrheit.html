  • Kunden glücklich und gebunden

  • Entscheidende Properties fehlen
  • "Stadtauto?" → halber Bestand

04 — Die Chance

  • Chatbasierte Suche statt Filter
  • Zusammenfassen, Orientierung geben
  • Standardfragen vorgefertigt beantworten
  • Nicht ersetzen, sondern abkürzen

05 — Die Gefahren

  • 59 % / 47 % „weiß ich nicht“
  • Injection: „Bau mir eine Website“ → Bot liefert HTML + PHP
  • Pointe: endet mit „kein Video gefunden“
  • Alles loggen, auswerten und daraus lernen

06 — Ablauf für Erfolg

  1. Klare Ziele definieren
  2. Echte Use Cases sammeln
  3. Prüfen, nachpflegen, dann bauen
  4. Erst Fragen, dann Daten, dann Bot

07 — Positiver Nebeneffekt

  • Bessere Landingpages
  • Besseres Social Media
  • Höhere Conversion
  • Die Bot-Analyse verbessert die ganze Seite

08 — Fazit

  • KI erfindet keine guten Ergebnisse
  • Schlechte Daten bleiben schlecht
  • Bot greift erst, wenn der Mensch sonst nichts findet
  • Der Chatbot ist der letzte Schritt, nicht der erste
